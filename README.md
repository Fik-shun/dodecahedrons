## Q2. Animating


### Key Bindings

|   Key    |       Description        |
|----------|--------------------------|
|    W     |      Object: move y+     |
|    S     |      Object: move y-     |
|    D     |      Object: move x+     |
|    A     |      Object: move x-     |
|    E     |      Object: move z+     |
|    Q     |      Object: move z-     |
|    T     |  Obj: CW rotate about z  |
|    R     |  Obj: ACW rotate about z |
|    F     |  Cam: CW spin around obj |
|    G     | Cam: ACW spin around obj |
|    ↑     |      Cam: move up        |
|    ↓     |      Cam: move down      |
|    →     |      Cam: move right     |
|    ←     |      Cam: move left      |
|   Esc    |           Quit           |
